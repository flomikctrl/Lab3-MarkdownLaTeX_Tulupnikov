## Markdown - это лёгкий язык разметки, который используется для оформления README-файлов, документации, отчётов и даже научных текстов. Он прост в использовании, хорошо читается как в сыром виде, так и после рендеринга, и широко поддерживается GitHub, VS Code и многими платформами.

# Markdown находит широкое применение в различных сферах:
Веб-разработка.

Блоги и платформы для написания.

Электронные письма и форумы.

Статические сайты.

Документация.

Системы управления версиями.

Примечания и заметки,

Презентации.

# Разработчикам Markdown нужен для быстрого и простого написания документации, README-файлов, заметок, комментариев и сообщений, которые легко читать и форматировать в простом тексте, а затем преобразовывать в HTML, PDF или другие форматы без сложных редакторов. ## Alert-блок

> [!TIP]
> **Совет по использованию:** Для экранирования специальных символов в Markdown используйте обратный слэш `\`. Например, чтобы написать \*курсив\* без форматирования, напишите `\*курсив\*`.

## Escape-символы

* Заголовок без форматирования: `\# Это не заголовок`
* Звёздочка как текст: `\*Просто звёздочка\*`

### 1. Inline-формула
Энергия покоя определяется знаменитой формулой Эйнштейна: $E = mc^2$.

### 2. Блочная формула (Интеграл)
Ниже представлена формула определенного интеграла:

$$\int_{a}^{b} x^2 \,dx = \frac{x^3}{3} \Big|_a^b$$

### 3. Дополнительные конструкции из списка

* **Дробь:**
$$\frac{a}{b}$$

* **Корень:**
$$\sqrt{x^2 + y^2}$$

* **Греческие буквы:**
$$\alpha + \beta = \gamma$$

* **Степень и индекс:**
$$x_{i}^{2n}$$
